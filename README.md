# Antigravity Quota Quickcheck

A native VS Code extension that adds a quick and easy way to check your Antigravity Quota directly from the status bar.

## ✨ Features

- **Status Bar Integration**: Displays a seamless `🚀 Antigravity Quota` icon in your bottom right status bar.
- **Instant Quota Access**: Click the status bar icon to immediately view your remaining model requests, percentages, and refresh times.
- **Native Popup UI**: Uses a native VS Code dropdown menu, preventing the need to open new editor tabs and perfectly matching your active VS Code theme.

## 📦 Installation

To install the extension for your daily programming workflow:

1. Download the `antigravity-quota-quickcheck-0.0.1.vsix` file.
2. Go to the **Extensions view** (`Ctrl+Shift+X` or `Cmd+Shift+X`) in VS Code.
3. Click the **`...`** (Views and More Actions) menu in the top right corner of the Extensions panel.
4. Select **"Install from VSIX..."**
5. Navigate to your downloaded `.vsix` file and select it.

## 🚀 Usage

1. Once installed, look at the bottom right corner of your VS Code editor.
2. You will see the **`🚀 Antigravity Quota`** item in the status bar.
3. Click it to open the quota dropdown menu.
4. The popup will display all your available models, remaining requests, and when the quota refreshes. Click anywhere else or press `Esc` to close it.

## Privacy Policy

This extension does not collect any data from the user. It is a client-side extension that displays data that is already available to the user in the Antigravity extension.
